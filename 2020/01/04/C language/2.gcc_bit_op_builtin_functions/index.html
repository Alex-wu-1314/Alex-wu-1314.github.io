GCC bit operations 内建函数详解

Alex Wu
2020-01-04
C语言

前言

GCC 提供了一系列以 __builtin_ 为前缀的内建函数（Built-in Functions）。这些函数由编译器直接支持，在编译时通常会被展开为对应的硬件指令（如 x86 的 BSF、BSR、POPCNT 等），因此在执行效率上往往远优于等价的手写循环实现。在算法竞赛和底层工程开发中，熟练使用这些函数可以显著提升代码的运行速度与简洁性。

1. __builtin_ctz() / __builtin_ctzll()

功能： 返回参数的二进制表示中，末尾（低位）连续 0 的个数（Count Trailing Zeros）。

ctz 对应 int 类型，ctzll 对应 long long 类型。

常见应用：

  • 快速求一个整数的最低有效位（Lowest Set Bit，LSB）的位置
  • 枚举子集时配合 lowbit 操作使用

2. __builtin_clz() / __builtin_clzll()

功能： 返回参数的二进制表示中，前导（高位）连续 0 的个数（Count Leading Zeros）。

clz 对应 int 类型（32位），clzll 对应 long long 类型（64位）。

  • 快速计算 $\lfloor \log_2 n \rfloor$：31 - __builtin_clz(n)（n > 0）
  • 判断一个数是否为 2 的幂次方

3. __builtin_popcount() / __builtin_popcountll()

功能： 返回参数的二进制表示中 1 的个数（Population Count，也称汉明重量）。

popcount 对应 unsigned int，popcountll 对应 unsigned long long。

  • 状压 DP 中统计状态中已选元素个数
  • 计算两个二进制串的汉明距离：__builtin_popcount(a ^ b)

4. __builtin_parity() / __builtin_parityll()

功能： 判断参数的二进制表示中 1 的个数的奇偶性。若 1 的个数为偶数返回 0，为奇数返回 1。

常见应用：

  • 奇偶校验（Parity Check）
  • 某些异或相关的数论/组合问题

6. __builtin_sqrt() / __builtin_sqrtf()

功能： 分别是标准库函数 sqrt()（返回 double）和 sqrtf()（返回 float）的 GCC 内建版本，用于快速开平方根。

__builtin_sqrt(x) double double 双精度（64位浮点）
__builtin_sqrtf(x) float float 单精度（32位浮点）

__builtin_sqrt / __builtin_sqrtf 是标准库 sqrt / sqrtf 的内建等价版本。在开启优化（如 -O2）时，GCC 会将它们直接映射为硬件浮点指令（如 x86 的 sqrtsd / sqrtss），避免了函数调用开销，因此在大量计算时具有明显的速度优势。

精度说明： __builtin_sqrt 使用双精度浮点（double，64位），精度更高；__builtin_sqrtf 使用单精度浮点（float，32位），精度较低但在某些场景下运算更快，内存占用更小。请根据实际需求选择。

总结对比表

函数 功能 参数类型 注意事项
__builtin_ctz(n) 末尾 0 的个数 unsigned int n=0 时 UB
__builtin_ctzll(n) 末尾 0 的个数 unsigned long long n=0 时 UB
__builtin_clz(n) 前导 0 的个数 unsigned int n=0 时 UB
__builtin_clzll(n) 前导 0 的个数 unsigned long long n=0 时 UB
__builtin_popcount(n) 1 的个数 unsigned int 无
__builtin_popcountll(n) 1 的个数 unsigned long long 无
__builtin_parity(n) 1 的个数奇偶性 unsigned int 偶返回 0，奇返回 1
__builtin_parityll(n) 1 的个数奇偶性 unsigned long long 偶返回 0，奇返回 1
__builtin_ffs(n) 最低 1 的位置（从 1 起） int n=0 时返回 0
__builtin_ffsll(n) 最低 1 的位置（从 1 起） long long n=0 时返回 0
__builtin_sqrt(x) 双精度开平方 double 返回 double
__builtin_sqrtf(x) 单精度开平方 float 返回 float

注意事项

  1. 这些函数是 GCC 扩展，不是 C/C++ 标准的一部分。在非 GCC 编译器下无法使用，移植性需注意。
  2. 传入 0 给 ctz / clz 是未定义行为，使用前务必判断非零。
  3. 开启编译优化（-O2 或更高）时，这些函数的性能收益最为明显。
